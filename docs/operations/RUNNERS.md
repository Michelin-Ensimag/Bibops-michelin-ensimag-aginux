# New Script Runners

These wrappers keep existing behavior but provide a cleaner entrypoint layout.

## Dev

- `python scripts/dev/init_sqlite.py`
- `python scripts/dev/build_it_vector_db.py`
- `python scripts/dev/run_mcp_server.py`

## Benchmark

- `python scripts/benchmark/run_core.py`
- `python scripts/benchmark/run_pipeline.py`
- `python scripts/benchmark/run_mcp_tools.py`

## Racing

- `python scripts/racing/run_hub.py`
- `python scripts/racing/run_demo.py`
- `python scripts/racing/run_arena.py`

## Copilot

- `python scripts/copilot/test_api.py`
- `python scripts/copilot/run_agent_mcp.py`

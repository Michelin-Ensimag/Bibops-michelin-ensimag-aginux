|  | **Default settings for EcoPrint standard**<br> |
| --- | --- |

The new Ricoh printers have the following default settings to use the EcoPrint standard:

| **Job Type**<br> | **Locked Print**: You will need to enter a code on the Ricoh printer itself for the job to print.<br> |
| --- | --- |
| **Duplex**<br> | **Open to Left**: Your print will be generated using both sides of the page.<br> |
| **Color/Black and White**<br> | **Black and White**: When printing to the MPC4000 capable of color print, the default is black and white. Note that color print costs 10 times the cost of black and white, so please do not change this default unless necessary.<br> |

|  | **How to print with a Ricoh Printer and retrieve the print-outs**<br> |
| --- | --- |

In every application there is a “Print” menu option. You only need to select the Ricoh printer when you plan to print with it.

| *Steps*<br> | *Actions*<br> | *Illustrations*<br> |
| --- | --- | --- |
| **1**<br> | For example, in WORD select “**File**” and “**Print**”<br> | ![image](Pasted%20image.png)<br> |
| **2**<br> | By default, you should have a Ricoh Printer name.<br>**If the Ricoh printer is not indicated by default in the “Name” field**, you can select it in the dropping down list.<br>Click on the**Print**button to launch the printing.<br> | ![image](Pasted%20image_28.png)<br> |
| **3**<br> | Notice the Printer’s properties, the “**Locked Print**” is the default and unique option.<br>Once you install the driver you**MUST**provide an user id and password.<br> | ![image](Pasted%20image_47.png)<br> |
| **4**<br> | The printer will hold your printing job until you come to the printer to retrieve it.<br> |  |
| **5**<br> | Go to the Ricoh printer :<br>Press the**Printer button**on the left of the unit. (The “copy” function is probably active).<br>Choose “**Print Jobs**” on the bottom on the touch screen. A list of jobs by user id will be displayed.<br>**Find your print job in the queue by looking for your user id**. When you find your print job,**touch its name**on the screen then touch the “**Print**” button on the right.<br>**Enter your Password**set in the step b above on the copy keypad to release the print.<br>**Confirm**that you want to print.<br> | ![image](Pasted%20image_23.png)<br> |

|  | **How to change a printer password (Locked Print - User ID and password)**<br> |
| --- | --- |

| *Steps*<br> | *Actions*<br> | *Illustrations*<br> |
| --- | --- | --- |
| **1**<br> | When you (re-)install a printer, the default password is 0000. (*)<br>**If you want to ensure the confidentiality of your printed**documents,**do not forget to change this password.**<br> |  |
| **2**<br> | Notice the “**Locked Print**” is a default and unique option. Once you install the driver you**MUST**provide an user id and password. The printer will hold your printing job until you come to the printer to retrieve it.<br> | ![image](Pasted%20image_8.png)<br> |
| **3**<br> | To set the User and Password ID, click on :<br>(Menu File)>**Print**>Setup tab> click on “**Details**”<br> |  |
| **4**<br> | **Enter your Michelin ID number**(e.g starting with “N”, or “FX” etc.) as the User ID on the following pop-up window.<br> |  |
| **5**<br> | **Enter a 4 – 8****digit****number**as your**Password**that will be used to retrieve your future Ricoh printings.<br>Note: You can modify this password in the future as you wish. This only changes the password for the selected printer.<br> |  |
| **6**<br> | Click**OK**on the “Locked Print Details” window<br>Then**OK**on the printer properties window.<br>You can now click**OK**on the Print window to print the document to the Ricoh Printer.<br> |  |

*(*) This is the password you must type on the printer’s keypad before retrieving your printed document(s).*
